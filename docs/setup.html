Setup Intructions

  1. Download and unzip the framework package.
  2. Change the server user, password and database in the includes/configuration.php file.
  3. Change the folder root path and the software version.
  4. Create a new database for the project
  5. Use the contents monzey-framework/database-setup.sql for the first query.
  6. Create your software and use this documentation for reference!
