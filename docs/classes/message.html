Message Class

Class for individual user messages.

Object

Object array that contains a message information.

Variables:

  • id
  • send_date
  • sender_id
  • user_id
  • message
  • status (read/unread/deleted)

Public Methods

send_message()

Function to send a message to a user.

Usage:

Message::send_message($user_id);

Returns:

nothing.

recent_messages()

Function to search for recent message from the current logged in user.

Usage:

Message::recent_messages($number);

Returns:

  • Array with Message Objects.
