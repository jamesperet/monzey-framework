User Class

Class for handling registered users

Object

Object array that contains a users information.

Variables:

  • id
  • username
  • password
  • first_name
  • last_name
  • user_email
  • registration_date
  • bio
  • avatar

Public Methods

authenticate()

Function to authenticate a user. It checks the username and the password in the database to see if they match.

Usage:

User::authenticate($username, $password);

Returns:

  • If true returns a User Object.
  • Else it returns false.

username()

Returns the users username based on his session on the server.

Usage:

User::username();

Returns:

  • the users username
  • empty

full_name()

Returns the users full name.

Usage:

User::full_name();

Returns:

  • the users full name
  • empty

checkUser()

Check if a users data is valid for registration in the system.

Usage:

User:checkUser();

Returns:

  • If success passes to AddUser() function.
  • invalid-email-error: the email is already registered on the system or contains invalid characters.
  • invalid-username-error: the username is already registered on the system or contains invalid characters.

addUser()

Function to add a user to the system. It is triggered automatically by the checkUser() function.

Usage:

User::addUser($username, $password, $email, $firstname, $lastname);

Returns:

  • Profile Created
