Reference Class

Class for referencing files in other files.

Object

Object array that contains the reference information.

Variables:

  • id
  • reference_type
  • description
  • reference_object_type
  • reference_object_id
  • reference_url
  • script_id
  • owner_id
  • creation_date

Public Methods

create_reference()

Function to create a reference on a file.

Usage:

Reference::create_reference($reference_type, $reference_object_type, $reference_object_id, $script_id);

Returns:

  • Reference Object
  • Empty

find_references()

Function to find references from a file.

Usage:

Reference::find_references($file_id);

Returns:

  • Array of Reference Objects
  • Empty
